    # Switch 开关

    简介

    你可以通过设置v-model来达到独立控制开关，或者控制开关的状态，activeColor inactiveColor分别控制开启和关闭的颜色

    # 使用

    <template>
  <div id="app">
    <my-switch name="12" v-model="active1"></my-switch>

    <my-switch
      name="11"
      v-model="active2"
      activeColor="red"
      inactiveColor="green"
    ></my-switch>
  </div>
</template>

<script>
  export default {
    name: "App",
    data() {
      return {
        active1: false,
        active2: false,
      };
    },
  };
</script>
